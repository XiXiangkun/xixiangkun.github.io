刷题笔记

Posted on 2020-02-22 | In C++

考研失利，准备工作的刷题笔记内容

Notes

  • 下面是列出的一些陌生的知识点和提炼

    定义：

  1. c++不允许数组长度定义含有未知变量，是因为编译器的问题。所以使用动态数组或者vector容器解决问题。
  2. iostream是标准的头文件
  3. long long最大9223372036854775807
  4. 定义容器不要再后面加括号

输入输出：

  1. vector只能使用str.push_back(j);或cin>>str[i];在顶部进行push和pop，在内部操作的效率很低
  2. 动态数组结合getline输入时，需要cin.ignore();取消换行键的输入，会影响正常的输入结果，而且需要delete[] str; str = NULL;释放空间，所以采用vector更好
  3. 同一行两个输入的：cin>>n>>d;
  4. 带空格的输入，使用getline(cin,str);
  5. ‘’表示字符，””表示字符串
  6. 使用getline时，结合while，不要用for，会出错，很离谱。getline函数对空白符的处理：不忽略空白符，即把空白符也当成元素存储到string对象中，读取字符一直到换行符（回车），读取终止且丢弃换行符（换行符从输入流中丢去且不保存在string对象中）；cin函数对空白符的处理：读取并忽略非空白符之前所有的空白字符，直到再次遇到空白符，读取终止（该空白符仍留在输入流中）。所以cin后，多用一个getline读取换行符。
  7. cin的输入可以是一行，用空格分开，也可以是回车分割的多行

函数：

  1. string的str.substr(start,end);左闭区间，右开区间的切割函数
  2. vector的v.empty()判断是否为空，空返回true；v.size()返回长度
  3. C++的count函数count(num.begin(), num.end(), n)计算begin到end中n的个数
  4. sort(num.begin(),num.end());是对容器元素从小到大的排序
  5. vector<int> newnum(num.begin() + 2, num.end());重新获取新的容器
  6. newnum.erase(find(newnum.begin(), newnum.end(), num[0] + 1));erase是删除，注意不是remove，区别在网上搜索，注意string也是使用erase
  7. 上面代码中，find为vector函数，可以查找元素
  8. sort需要使用头文件algorithm
  9. string的find使用while (str_in.find(" ", index) != str_in.npos)利用index变换，查找每一个特定字符
  10. string的replace使用str_in = str_in.replace(num, 1, ",");起始位，替换长度，替换字符串
  11. 使用getline时，结合while，不要用for，会出错，很离谱。getline函数对空白符的处理：不忽略空白符，即把空白符也当成元素存储到string对象中，读取字符一直到换行符（回车），读取终止且丢弃换行符（换行符从输入流中丢去且不保存在string对象中）；cin函数对空白符的处理：读取并忽略非空白符之前所有的空白字符，直到再次遇到空白符，读取终止（该空白符仍留在输入流中）。所以cin后，多用一个getline读取换行符。
  12. 借助头文件algorithm使用sort(vec.begin(),vec.end());可以将容器排序
  13. 使用erase函数：arr.erase(arr.begin());
  14. 定义二维数组可以使用vector<vector<int>> arr(2, vector<int>(2));整数默认值为0

名词：

  1. STL标准模板库，包括迭代器、算法、容器、仿函数、内存配置器和配接器
