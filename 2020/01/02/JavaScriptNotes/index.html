JavaScript补充笔记，HTML描述网页、CSS定义样式、JavaScript是脚本语言。

Notes

  • script标签的使用，JavaScript 代码必须位于 script标签之间。script可以放置在head或者body中，脚本可以放置在外部文件中。

  • getElementById()查找特定id的元素，修改它的元素内容(可以是文字null到有、更换图片、更改样式等)

  • JavaScript显示方案：

    • innerHTMl：是在 HTML 中显示数据的常用方法。
    • document.write()写入 HTML 输出。HTML 文档完全加载后使用 document.write()将删除所有已有的 HTML ，例如使用按钮。仅用于测试
    • window.alert()：能够使用警告框来显示数据
    • console.log()：在浏览器中使用 console.log() 方法来显示数据。
  • 区分null和undefined
  • JS函数中，toCelsius 引用的是函数对象，而 toCelsius() 引用的是函数结果。
  • HTML事件：onchange、onclick、onmouseover、onnouseout、onkeydown、onload

一些特别的知识点：

  • hoisting：将所有声明提升到当前作用域顶部的默认行为
  • "use strict"; 定义 JavaScript 代码以“严格模式”执行，不允许自动提升声明
  • 在 HTML 事件处理程序中，this 指的是接收此事件的 HTML 元素
  • 在块{ }中使用let代替var声明变量，可以解决在块中修改var而改变全局变量的问题

JavaScript函数

  • call可以调用所有者对象（不同对象）作为参数
  • apply把自己的函数应用于别的对象
  • JavaScript闭包：JavaScript中的函数会形成闭包。闭包是由函数以及创建该函数的词法环境组合而成，这个环境包含了这个闭包创建时所能访问的所有局部变量。所以里面的局部变量仍然可用。

HTML DOM（文档对象模型）

  • 通过方法和属性来改变HTML内容，最简单的就像上面的 getElementById 是方法，而 innerHTML 是属性。
  • setInterval( )函数按照指定的周期（以毫秒计）来调用函数或计算表达式
  • 通过addEventListener( )函数绑定两个要执行的函数

HTML BOM（浏览器对象模型）

  • 就是对浏览器的一系列操作函数
  • 浏览器端口，http默认80，https默认443
  • 弹窗、time定时功能、Cookies的应用

AJAX

  • 不刷新页面更新网页，页面加载后从服务器请求和接收数据，后台向服务器发送数据。
  • 不是编程语言，只是组合了XMLHttpRequest对象、JavaScript和HTML DOM。
    • 发生某个事件，创建XMLHttpRequeat发送HttpRequest
    • 服务器处理HttpRequest，创建响应返回
    • JS处理响应，更新页面

JSON:

  • JavaScript Object Notation（JavaScript 对象标记法），是一种存储和交换数据的语法。
  • JSON字符串必须使用双引号
  • JSON优于XML
  • 解析：JavaScript 函数 JSON.parse() 把文本转换为 JavaScript 对象
  • 字符串化：通过 JSON.stringify() 把 JavaScript 对象转换为字符串

jQuery

  • jQuery 一直是世界上最受欢迎的 JavaScript 库。
  • 通过 id 来查找 HTML 元素var myElement = $("#id01");
  • 通过标签名来查找 HTML 元素var myElements = $("p");
  • 通过类名来查找 HTML 元素var myElements = $(".intro");

  • 通过 CSS 选择器查找 HTML 元素，返回包含 class=”intro” 的所有p元素的列表。var myElements = $("p.intro");

  • 设置 HTML 元素的内部文本：myElement.text("Hello China!");

  • 获取 HTML 元素的内部文本：var myText = myElement.text();
  • 设置元素的 HTML 内容：var myElement.html("<p>Hello World</p>");
  • 获取元素的 HTML 内容：var content = myElement.html();
  • 隐藏一个 HTML 元素：myElement.hide();
  • 显示一个 HTML 元素：myElement.show();
  • 更改 HTML 元素的字体尺寸：myElement.css("font-size","35px");
  • 删除 HTML 元素：$("#id").remove();
  • 返回 HTML 元素的父元素：var myParent = myElement.parent();
  • 在W3school往回走那上有许多实例可以使用
