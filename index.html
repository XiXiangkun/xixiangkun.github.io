JAVA_Reflection+Annotation

Reflection

1. 反射，用来通过对象获取类

　　实例：
　　import java.lang.reflect.Method
　　Class<?> c = null; //声明对象
　　c = Class.forName(“org.lxh.demo.SimpleOne”); //实例化
　　Method toM = c.getMethod(“toString”); //取得toString方法
　　toM.invoke(c.newInstance()); //invoke()进行调用

  • Class.newInstance()用来反射无参的构造器
  • Constructor.newInstance()可以反射任何构造器

2. invoke()方法：调用包装在当前Method对象中的方法

　　参数（obj：实例化后的对象，args：用于方法调用的参数）
　　有参数的情况：
　　Method met = c.getMethod(“sayHello”,String.Class,int.Class);
　　String rv = null;
　　rv = (String)met.invoke(c.newInstance,”xxk”,21);　　

Annotation

1. @override：可以用来检查是否正确重写

2. @interface：定义一个Annotation

　　public @interface MyDefaultAnnotationValiue{
　　　　public String key() default “XXK”;
　　}

  • 变量后面要加()，可以设置默认值

3. @Target：指定Annotation使用范围

4. @Documented：生成java.doc时编写文档信息

5. 通过反射获得Annotation（反射与Annotation结合才有意义）
