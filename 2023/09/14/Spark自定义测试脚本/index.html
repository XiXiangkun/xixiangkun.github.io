SparkиҮӘе®ҡд№үжөӢиҜ•и„ҡжң¬

SparkиҮӘе®ҡд№үжөӢиҜ•и„ҡжң¬пјҲеҹәдәҺYahooзҡ„Streaming-benchmarksпјү

  • дҪҝз”ЁYahooзҡ„жөӢиҜ•е·Ҙе…·дёӯйҒҮеҲ°еҫҲеӨҡй—®йўҳпјҢй…ҚзҪ®ж–Үд»¶дёҚз”ҹж•ҲгҖҒеўһеҠ еҲҶеҢәж•°еҜјиҮҙж¶Ҳиҙ№иҖ…з©әи·‘зӯү
  • жүҖд»ҘеҹәдәҺYahooе·Ҙе…·зҡ„йҖ»иҫ‘йҮҚж•ҙдәҶжөӢиҜ•и„ҡжң¬пјҢйҮҚзӮ№е…іжіЁZookeeperгҖҒKafkaгҖҒSparkгҖҒRedisжҗӯе»әзҡ„еӨ§ж•°жҚ®еңәжҷҜзҡ„жҖ§иғҪ

1. Yahooзҡ„е·Ҙе…·йҖ»иҫ‘

1.1 RedisгҖҒKafkaж•°жҚ®жіЁе…Ҙ

  • жңүзӣёе…іжңҚеҠЎзҡ„еҗҜеҠЁ
  • ж•°жҚ®жіЁе…ҘRedisпјҢйў„е…ҲжіЁе…ҘRedisзҡ„ж•°жҚ®пјҢlein run -n --configPath ../conf/benchmarkConf.yaml
    • й”®пјҡad_idпјҢеҖјпјҡcampaign_id
  • еҲӣе»әTopicпјҢй»ҳи®Өad-eventsпјҢ1еүҜжң¬пјҢ1еҲҶеҢә
  • SparkеҗҜеҠЁж¶Ҳиҙ№
  • еҶҷе…ҘKafkaж•°жҚ®пјҢдҪҝз”ЁLeiningenиҮӘеҠЁеҢ–ClojureйЎ№зӣ®з®ЎзҗҶпјҢlein run -r -t 1000 --configPath ../conf/local.yaml
    • е№ҝе‘ҠдәӢд»¶жҳҜйҡҸжңәзҡ„пјҢе№ҝе‘ҠеҜ№еә”зҡ„жҙ»еҠЁйңҖиҰҒеҲ°RedisдёӯиҺ·еҸ–
    • data/src/setup/core.cljзҡ„жәҗз ҒдёӯпјҢcampaignsй»ҳи®Өдёә100пјҢеҸӮж•°-tи®ҫзҪ®жҜҸз§’дәӢд»¶ж•°дёә1000пјҢadsдёә1000
    • еҶҷе…ҘKafkaзҡ„ж•°жҚ®пјҢеқ‘пјҡYahooзҡ„д»Јз ҒдёӯеҶҷжӯ»дәҶtopicдёәad-events
      • use_id
      • page_id
      • ad_idпјҡд»ҺadsдёӯйҡҸжңәйҖүжӢ©е№ҝе‘ҠID
      • вҖҰ
      • event_typeпјҡ[вҖңviewвҖқ, вҖңclickвҖқ, вҖңpurchaseвҖқ]

1.2 Yahooзҡ„START_DSTREAM_SPARK_PROCESSINGйҖ»иҫ‘

1.2.1 иҜ»еҸ–й…ҚзҪ®ж–Үд»¶гҖҒи®ҫзҪ®TopicйӣҶгҖҒеҲӣе»әDStream
graph TD
A(Producer)--Advertisments-->B(Kafka)
B--DStream-->C(Spark)
  • иҜҘжөӢиҜ•йЎ№дёӯпјҢSparkдҪҝз”ЁDStreamиҝӣиЎҢж•°жҚ®еӨ„зҗҶпјӣDStreamдёәSparkзҡ„жү№еӨ„зҗҶжЁЎеһӢ
  • DStreamзҡ„еҜ№иұЎйҖҡиҝҮforeachRDDеҮҪж•°еҜ№ж•°жҚ®жү№иҝӣиЎҢеӨ„зҗҶпјҢRDDдёәbatch.timeж—¶й—ҙй—ҙйҡ”еҶ…Kafkaж”¶йӣҶеҲ°зҡ„ж•°жҚ®

1.3 ж•°жҚ®еӨ„зҗҶ

1.3.1 жҸҗеҸ–valueгҖҒи§ЈжһҗдёәJson
  • й’ҲеҜ№жҜҸжқЎж¶ҲжҒҜиҝӣиЎҢеӨ„зҗҶ
  • е®Ңж•ҙзҡ„ж¶ҲжҒҜеҢ…жӢ¬KeyгҖҒValueгҖҒTopicгҖҒPartitionзӯүпјҢеҸӘжҸҗеҸ–Value
1.3.2 иҝҮж»Өж•°жҚ®гҖҒж•°жҚ®жҠ•еҪұ/еҲҮеүІ

  • дҝқз•ҷж—¶й—ҙзұ»еһӢдёәviewзҡ„ж•°жҚ®

  • еҸӘдҝқз•ҷad_idгҖҒevent_time

1.3.3 RedisжҹҘиҜў
  • жҹҘиҜўRedisпјҢиҝ”еӣһ[campaign_id, ad_id, event_time]

1.3.4 ж•°жҚ®иҪ¬жҚўгҖҒж•°жҚ®иҒҡеҗҲ

  • ж—¶й—ҙжҲіиҲҚе…ҘпјҢиҪ¬дёә((campaign_id, window_time), ad_id)

  • е…ғзҙ еҖјad_idжҳ е°„дёә1пјҢReduceByеҮҪж•°иҝӣиЎҢжҢүй”®иҒҡеҗҲ

1.3.5 еҶҷе…ҘRedis
  • жҢүRDDжү№еӨ„зҗҶпјҢиҝҷйҮҢUUIDеҸҜд»ҘзҗҶи§Јдёәе“ҲеёҢиЎЁеҗҚ/еҲ—иЎЁеҗҚ
  • seen_countжҳҜзӘ—еҸЈж—¶й—ҙеҶ…е№ҝе‘Ҡзҡ„жҹҘзңӢж¬Ўж•°пјӣtime_updatedжҳҜseenжӣҙж–°еҗҺзҡ„жңҖеҗҺдёҖж¬Ўж—¶й—ҙ

2. йҮҚж•ҙеҗҺзҡ„й“ҫи·Ҝ

2.1 зҺҜеўғ

  • 58пјҡZK
  • 60пјҡSpark
  • 65пјҡRedisгҖҒLein
  • 66пјҡKafka

2.2 йҖ»иҫ‘

2.2.1 еҮҶеӨҮ
  • зӣёе…іжңҚеҠЎзҡ„еҗҜеҠЁжӯҘйӘӨ
    • ZKжӯЈеёёеҗҜеҠЁ
    • KafkaдҪҝз”Ё3 brokerпјҢйңҖиҰҒ3д»Ҫй…ҚзҪ®ж–Үд»¶пјҢдҝқиҜҒbroker idгҖҒportгҖҒlog dirдёҚеҶІзӘҒ
    • SparkжӯЈеёёеҗҜеҠЁпјҢжөӢиҜ•дҪҝз”Ёзҡ„Kafka IPгҖҒPortйңҖиҰҒSetupзј–иҜ‘еҗҺжүҚиғҪз”ҹж•Ҳ
    • RedisжӯЈеёёеҗҜеҠЁ
2.2.2 жөӢиҜ•

  • зЎ®и®ӨRedisз«ҜжІЎжңүleiningen.core.mainзӣёе…іиҝӣзЁӢпјҲеҲ йҷӨTopicд№ӢеүҚйңҖиҰҒе…ій—ӯRedisжңҚеҠЎпјҢеҗҰеҲҷеҲ йҷӨеҠЁдҪңдјҡжІЎжңүж•Ҳжһңпјү

  • еҲ йҷӨдёҠж¬ЎжөӢиҜ•дҪҝз”Ёзҡ„TopicпјҢеӣ дёәYahooжәҗз Ғдёӯз»‘е®ҡдәҶTopicеҗҚдёәad-eventsпјҢжүҖд»ҘдёҚж–№дҫҝдҪҝз”Ёе…¶е®ғTopicиҝӣиЎҢжөӢиҜ•

    • ./bin/kafka-topics.sh --delete --bootstrap-server 10.10.31.58:9092 --topic ad-events

  • йҮҚеҗҜRedis

  • йҮҚж–°еҲӣе»әTopic

    • ./bin/kafka-topics.sh --create --bootstrap-server 10.10.31.58:9092 --replication-factor 3 --partitions 5 --topic ad-events

  • ж•°жҚ®жіЁе…ҘRedisпјҢйў„е…ҲжіЁе…ҘRedisзҡ„ж•°жҚ®пјҢlein run -n --configPath ../conf/benchmarkConf.yaml

  • SparkеҗҜеҠЁж¶Ҳиҙ№пјҲжӯӨж—¶и§ӮеҜҹж¶Ҳиҙ№зҠ¶жҖҒпјҢжҳҜжІЎжңүж•°жҚ®жөҒзҡ„пјҢwatch -d -n 1 "./bin/kafka-consumer-groups.sh --bootstrap-server 10.10.31.58:9092 --all-groups --describe"пјү

  • еҶҷе…ҘKafkaж•°жҚ®пјҢдҪҝз”ЁLeiningenиҮӘеҠЁеҢ–ClojureйЎ№зӣ®з®ЎзҗҶпјҢlein run -r -t 1000 --configPath ../conf/local.yamlпјҢжӯӨж—¶ж¶Ҳиҙ№зҠ¶жҖҒеҸҜд»Ҙи§ӮеҜҹеҲ°ж•°жҚ®жөҒпјҢLOG-END-OFFSETгҖҒLAGжҳҜеҸҳеҢ–зҡ„

  • иҝҷйҮҢеҮәзҺ°иҝҮдёҖдёӘзҺ°иұЎпјҢжөӢиҜ•е®ҢжҲҗеҗҺпјҢеҰӮжһңдёҚkillжҺүRedisз«Ҝзҡ„leiningen.core.mainзӣёе…іиҝӣзЁӢе’ҢRedisжңҚеҠЎпјҢдјҡзңӢеҲ°ж¶Ҳиҙ№зҠ¶жҖҒдёҖзӣҙжҳҜеҸҳеҢ–зҡ„пјҢLOG-END-OFFSETгҖҒLAGиҮӘеўһеҲ°дәҶеҫҲжҒҗжҖ–зҡ„ж•°еҖј

2.3 з»“жһң
