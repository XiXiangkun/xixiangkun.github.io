      • 梯度下降：
        • 特征缩放：标准化、归一化

    4. TensorBoard可视化：

      • Tensorboard可以记录与展示以下数据形式：标量 Scalars、图片 Images、音频 Audio、计算图 Graph、数据分布 Distribution、直方图 Histograms、嵌入向量 Embeddings
      • Tensorboard的可视化过程
        • 建立一个graph,想从graph中获取某些数据的信息
        • 确定要在graph中的哪些节点放置summary operations以记录信息
        • 使用 tf.summary.scalar 记录标量
        • 使用 tf.summary.histogram 记录数据的直方图
        • 使用 tf.summary.distribution 记录数据的分布图
        • 使用 tf.summary.image 记录图像数据
        • operations并不会去真的执行计算，除非run,或者它被其他的需要run的operation所依赖。
        • 特地去运行所有的summary节点。一份程序下来可能有超多这样的summary 节点，要手动一个一个去启动极其繁琐，可以使用tf.summary.merge_all去将所有summary节点合并成一个节点，只要这个节点，产生所有之前设置的summary data。
        • 使用tf.summary.FileWriter将运行后输出的数据都保存到本地磁盘
        • 运行整个程序，并在命令行输入运行tensorboard的指令，之后打开web端查看可视化的结果

    1. WX+B：

      • 神经网络一般采用wx+b或xw+b，每个小x都是一个d维的列向量，w权重，b偏置。得到的得分最高代表预测结果，用损失函数判断结果损失，想办法使损失最小。

    2. 收敛器：

      • Adadelta最快，但复杂；Adagrad折中推荐使用；SGD最慢：查看论文image credits ti Alec …

    3. 全连接的ReLU神经网络项目实践流程：

      1. 数据集导入：转成numpy数组，数据集划分（训练、验证、测试），归一化数据（-平均/标准差）
      2. 将训练集（验证和测试）转成数据集类，调用enumerate时使用迭代器，将数据集转成小部分
      3. 设置GPU使用
      4. 训练模型：重置默认图，设置x、y参数为placeholder，设置权重参数w1、w2（Kaiming的归一化方法初始化），计算h，判断得分返回
      5. 计算损失函数，修改相应权重（identity保证op生效）
      6. 创建会话，初始化
      7. 循环开始训练，for t, (x_np, y_np) in enumerate(train_dset):配合迭代器执行
      8. 设置feed_dict，tf.Session.run()执行fetches中的操作，计算fetches中的张量值
      9. 打印输出准确率和损失

    4. 卷积：

      1. 上述问题：flatten函数使空间信息缺失。
      2. 解决：使用卷积函数CONV调整w（3*3的w将每一维5*5的x卷积成3*3，三维拼合）,卷积完成激活RELU，比0大的留下，小的删去。
      3. 卷积输出：使用pad保证大小不变，N个卷积和保证输出10： 例如，32*32*3经过10个5*5卷积层，stride 1， pad 2：得到(32+2*2-5)/1+1=32，所以输出32*32*10
      4. 池化层POOL：让图变小，池化步长没有1，步长为2则从2*2中找到最大的。
      5. 参考网址：
        • http://cs231n.github.io/convolutional-networks/
        • https://www.cnblogs.com/bernieloveslife/p/10197063.html
