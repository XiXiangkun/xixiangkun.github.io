  • 机器学习（Python–Scikit-learn）：

    1. 机器学习流程：

      • 问题定义（分类、回归、聚类）
      • 数据准备
      • 模型训练
      • 模型评估
      • 模型部署
      • 模型监控与更新

    2. 探索性数据分析EDA：

      • 数据集整体描述、单变量分析+直方图表示、分类型变量、多变量分析+相关性矩阵+热力图绘制
      • 特征抽取：
        • 文本型：
          • TF-IDF（词频-逆向文件频率）
          • Word2vec（词向量）
        • 时间日期：
      • 特征转换：
        • 标准化、归一化、对数变换、特征离散化、独热码等预处理
      • 特征选择：
        • Filter：赋权评分
        • Wrapper：GA遗传算法
        • Embeded：随机森林

    3. 算法介绍：

      • 逻辑回归应用在神经网络：
      • 决策树：每个节点代表一个属性，根据属性划分；运用熵和信息增益的知识
      • 集成学习Ensemble Learning：
        • Bagging：随机有放回抽样（随机森林）
        • Boosting：组合时有权重（Gradient Boosting、GBM）

    4. 超参数寻优（用于评估）

      • 方法：
        • 网格搜索：经验确定范围，范围内穷举
        • 随机搜索：
      • 交互检验：
        • K折交互检验
        • 时间序列数据的交互检验（保证验证机在训练集后面）
      • Scikit-learn超参数寻优方法：
        • GridSearchCV()：基于交互检验和网格搜索
        • RandomizedSearchCV()：基于交互检验和随机搜索
      • GBM超参数寻优：
        • 影响梯度推进类超参数：
          • learnning_rate学习速率
          • n_estimators推进的梯度数量
        • 影响树结构类超参数：
          • max_depth控制单个决策树的最大深度
          • max_features允许单个决策树在每个节点上使用特征的最大数量
          • min_samples_split单个决策树内部节点再划分所需最小样本数
          • min_samples_leaf叶子节点最小的样本数量
        • 随机性相关的超参数：

      • 基础：Python
      • 进阶：Numpy、Scipy、Pandas
      • 机器学习：Sklearn、TensorFlow
