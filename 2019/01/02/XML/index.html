W3C标准的笔记内容之一（一）

Notes

  • W3C推荐标准
  • 可扩展标记语言（EXtensible Markup Language）
  • 设计宗旨是传输数据，需要自行定义标签
  • XML应用于web开发，把数据分离，就不会发生改变数据花费大量时间的问题
  • XML 中，省略关闭标签是非法的。所有元素都必须有关闭标签
  • XML 的属性值须加引号
  • 用实体引用来代替 “<” 字符
  • 注释的使用：
  • XML 中，文档中的空格不会被删节

  • 通过DTD验证XML文档的结构

  • CSS可以格式化XML，但是不常用，推荐XSLT

  • XMLHttpRequest和JS的AJAX联合实现页面加载

  • 加载XML解析器：

  • 在 XHTML 中，xmlns 属性是必需的。在 HTML 中，xmlns 属性是无效

  • XML的xmlns属性定义命名空间，用来区分DTD中不同的结构要求

##CDATA：

  • CDATA 指的是不应由 XML 解析器进行解析的文本数据

  • CDATA 部分由 “<![CDATA[“ 开始，由 “]]>” 结束：

  • PCDATA 指的是被解析的字符数据
