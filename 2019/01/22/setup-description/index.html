зҪ‘дёҠзҡ„ж•ҷзЁӢпјҢиҜҙд»Җд№ҲжңҖиҜҰз»ҶпјҢйғҪжҳҜйӘ—дәәзҡ„

### еҶ…е®№пјҡ

  • е®үиЈ…GitпјҢдёӢдёҖжӯҘеҲ°еә•
  • е®үиЈ…NodejsпјҢдёӢдёҖжӯҘеҲ°еә•
  • дҪҝз”Ёе‘Ҫд»ӨиЎҢcmdе®үиЈ…hexoпјҢзҪ‘дёҠжңүиҜӯжі•пјҲжіЁж„Ҹе…ЁеұҖй…ҚзҪ®пјҢж·»еҠ зҺҜеўғеҸҳйҮҸзӯүзҡ„й—®йўҳпјҢжңүnpmзҡ„дёҖдёӘеҲ·ж–°е‘Ҫд»ӨпјүгҖӮ
  • дёҖдәӣе®үиЈ…ж“ҚдҪңеҗҺпјҢдҪҝз”Ёhexo -vжҹҘзңӢзүҲжң¬
  • е…іиҒ”Githubе’ҢиҮӘе·ұhexoжҗӯе»әзҡ„еҚҡе®ўпјҢжіЁж„ҸжҹҘзңӢзҪ‘дёҠж•ҷзЁӢе°ұеҸҜд»Ҙ
  • еңЁж №зӣ®еҪ•(hexo_try)дёӢпјҢеҸій”®git bashиҝӣиЎҢйғЁзҪІпјҲиҝңзЁӢйғЁзҪІе‘Ҫд»Өпјҡhexo cжё…йҷӨзј“еӯҳпјӣhexo g + hexo dпјү

### жіЁж„Ҹпјҡ

  1. дҪҝз”Ёhexoзҡ„иҝҮзЁӢдёӯпјҢ_config.ymlдёӯзҡ„пјҡеҗҺйқўпјҢдёҖе®ҡиҰҒеҠ з©әж јпјҢеҗҰеҲҷдёҚдјҡжҠҘй”ҷпјҢдҪҶжҳҜдёҚдјҡйғЁзҪІжҲҗеҠҹ
  2. еҺ»githubеҲӣе»әд»“еә“пјҢдёҺиҮӘе·ұзҡ„з”ЁжҲ·еҗҚдёҖиҮҙпјҢз”ЁжқҘиҝңзЁӢйғЁзҪІд»“еә“пјҢ_config.ymlдёӯдҝ®ж”№
    гҖҖгҖҖdeploy:
    гҖҖгҖҖгҖҖгҖҖtype: git
    гҖҖгҖҖгҖҖгҖҖrepo: https://github.com/XiXiangkun/XiXiangkun.github.io.git
    гҖҖ гҖҖгҖҖгҖҖbranch: master
  3. hexo serverз”ЁжқҘеҺ»localhost:4000жҹҘзңӢиҮӘе·ұеҪ“еүҚзҡ„з»“жһң
  4. иҝңзЁӢйғЁзҪІе‘Ҫд»Өпјҡhexo -cжё…йҷӨзј“еӯҳпјӣhexo -g + hexo -d
  5. еҸҜиғҪйңҖиҰҒзҷ»еҪ•githubзҡ„зӘ—еҸЈжҸҗзӨә
  6. sshеҜҶй’Ҙзҡ„й…ҚзҪ®пјҢзҪ‘дёҠжңү
  7. зҷ»еҪ•xixiangkun.github.ioе°ұеҸҜд»ҘжҹҘзңӢеҚҡе®ў
  8. е®үиЈ…nodejsеҸӘжҳҜдёәдәҶз”Ёnpmе®үиЈ…hexo
  9. hexoеҸ‘йҹі hack so
  10. cdе‘Ҫд»ӨеҲҮжҚў cd /d D:\software
  11. еҲ йҷӨgithubдёҠзҡ„д»“еә“пјҢзӮ№ејҖеҗҺиҝӣеҺ»setting
