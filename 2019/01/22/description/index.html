踩过的坑

网上的教程，说什么最详细，都是骗人的

内容

  • 安装Git，下一步到底
  • 安装Nodejs，下一步到底
  • 使用命令行cmd安装hexo，网上有语法（注意全局配置，添加环境变量等的问题）。
  • 一些安装操作后，使用hexo -v查看版本
    1.使用hexo的过程中，_config.yml中的：后面，一定要加空格，否则不会报错，但是不会部署成功
    2.去github创建仓库，与自己的用户名一致，用来远程部署仓库
    3.hexo server用来去localhost:4000查看自己当前的结果
    4.远程部署命令：hexo -c清除缓存；hexo -g + hexo -d
